\section{Principles}

{\bf Referential transparency} \index{Referential
transparency}\cite{referential-transparency} -- an expression is called
referentially transparent if it can be replaced with its corresponding value
(and vice-versa). This requires that hte expression be {\bf pure} \index{Pure
expression}:

\begin{itemize}
    \item Its value must be the same for the same inputes;
    \item Its evaluation must have no side effects.
\end{itemize}
